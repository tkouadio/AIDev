# Projet Final – MGL869 (ÉTS)
Modèle prédictif pour analyser l’acceptation des Pull Requests générées par agents IA dans le dataset AIDev (MSR 2026).

## Structure
- `scripts/` : scripts Python (train_model.py, evaluate_model.py…)
- `notebooks/` : notebooks d’analyse exploratoire
- `data/` : non inclus (cf. Zenodo)
- `artifacts/` : modèles générés (non inclus)

## Installation
python -m venv .venv
source .venv/Scripts/activate
pip install -r requirements.txt
